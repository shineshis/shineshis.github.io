## 记录一些常用的正则表达式

### 整数或者一位小数,适用于精度要求一位的

```
regex = /^\d{0,5}([\b]*|\.|\.\d{0,1}|$)$/
```

> 其中 {0，5}指定整数位的长度，即0-99999，{0,1}指定小数位的长度，即x.0-x.9

### 金币正则

```
regex = /^\d{0,6}([\b]*|\.|\.\d{0,2}|$)$/
```

> 表示精确到分

### 单词正则

> 在一篇文章中做段词等，可以分词预处理等一些场合都可以使用

```
regex = /[a-zA-Z]+/
```
